Getting Started

Go from zero to hero with Pandakit

Preface

Using Pure CSS and JavaScript, Pandakit was built so you can easily extend, read, and understand, and beginners can easily contribute without having to learn or setup servers and more to get started. The easiest way to install Pandakit is by downloading zip file or cloning git repo which contains special scripts and style files to make setup seamless. For installing with specific integrations like Ember, React, or Meteor, see our integration guide For integrating Pandakit tasks into your own build tools, or using a CDN see our recipes section.
